blank-canvas-mark
=================

benchmarks for blank canvas

## To run Haskell

    $ cabal build
    $ ./dist/build/blank-canvas-mark/blank-canvas-mark -o foo.html -u foo.csv

## To run JavaScript

### OS X
    $ cd js
    $ open benchmark.html

### Linux/BSD
    $ cd js
    $ xdg-open benchmark.html
